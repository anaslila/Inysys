About Invsys Manager

Invsys Manager

Version 1.03

Complete inventory, customer, sales, and invoice management system with a classic Windows 95 interface. Fully keyboard accessible with advanced search capabilities and stock management.

New in v1.03:

  • SVG Icons from svgrepo integrated
  • Fixed quantity field backspace navigation
  • Auto-club duplicate products in invoice
  • Empty default quantity field
  • Tab/Enter to add and return to product field
  • Removed default tax rate from settings
  • Fixed bills history view
  • Improved table selection functionality
  • Auto-refresh after adding products/customers
  • New loading animation
